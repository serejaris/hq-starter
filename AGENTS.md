# AGENTS.md

Обновлено: YYYY-MM-DD

Эта папка — личный HQ. Минимальный набор файлов, по которым можно
быстро понять контекст: кто пользователь, что у него сейчас в работе,
какие длинные проекты идут параллельно.

## Как читать HQ

1. Сначала `me.md` — устойчивый контекст: кто пользователь, чем
   занимается, какой стек, какая аудитория, какие продукты.
2. Потом `now.md` — что в активной работе сейчас, ближайшие шаги,
   открытые вопросы.
3. Если задача связана с конкретным направлением — открой
   соответствующий файл в `projects/`.

## Правила работы

- Не выдумывай факты, метрики, даты. Если данных в HQ нет — скажи прямо,
  чего не хватает.
- Один факт — один файл-владелец. Не дублируй одну и ту же информацию
  в нескольких файлах.
- Если ответ зависит от свежести данных, посмотри дату сверху файла.
- Сначала вывод, потом детали.
- Для задач — следующие действия, а не общие советы.

## Где что лежит

| Файл | Роль |
|---|---|
| `me.md` | Устойчивый контекст: кто, чем занимается, стек, аудитория. |
| `now.md` | Текущий фокус: задачи недели, ближайшие шаги, открытые вопросы. |
| `projects/` | Длинные активные проекты — по файлу на каждый. |
| `prompts/` | Готовые промпты для заполнения и обслуживания HQ. |
